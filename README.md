# Glossolalia Dial

Type a sentence. Pick a voice. Turn the dial 0 → 4.

At **0** you hear the sentence read cleanly. At **2** you hear it half-dissolved — recognizably the same shape, the same voice, but lexically blurred. At **4** you hear wordless tongues that *sound* like a real language but aren't, still in the same speaker.

A **Morph** button sweeps 0 → 4 across one continuous take.

Built for the HuggingFace + Gradio **Build Small Hackathon**, track 🍄 *An Adventure in Thousand Token Wood*.

## How it works

A single fine-tuned LoRA on F5-TTS (~336M params) learns a control token — `tongues zero..four` — that grades the intelligibility of the same input sentence in the same voice. The trick is the training data: 500 sentences × 5 levels × 3 voices, where each clip is generated by base F5-TTS reading a *corrupted* version of the source sentence (g2p → CMUdict-bigram phoneme substitution at `p ∈ {0, 0.25, 0.5, 0.75, 1.0}`, syllable count + stress preserved). The LoRA learns to map (original sentence + level) → (corrupted-sounding audio in the same voice).

See [`BLOG.md`](BLOG.md) for the full Field Notes, and [`IDEA-AUDIT.md`](IDEA-AUDIT.md) for what we killed along the way.

## Validation gates (all must pass before shipping)

- Whisper-WER monotonic across levels 0..4, Spearman ≥ +0.80
- Resemblyzer cosine vs level-0 reference ≥ 0.85 (voice preserved)
- Perceptual: dial=2 wavs are a partial dissolution, not bimodal collapse

```bash
pip install -r requirements.txt
python app_server.py     # v2 custom-knob UI (the submitted entrypoint)
# or
python app.py            # v1 gr.Blocks UI with a gr.HTML knob
```

## Constraints honored

- ≤ 32B params (F5-TTS is ~336M).
- Gradio + HF Space, no cloud APIs.
- Open weights, open data, public dataset corpus (LibriTTS-R / LibriSpeech, public domain + CC-BY-4.0).
